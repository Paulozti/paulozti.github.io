# 🎮 Shivary Game Studio - Official Website

Este é o repositório oficial do site do **Shivary Game Studio**, desenvolvido por Paulo. O site serve como o hub central do estúdio, apresentando nosso portfólio de jogos na Steam, informações de contato e press kit.

**🌐 Acesse em:** [shivarygamestudio.com](https://shivarygamestudio.com)

---

## 🛠️ Tecnologias Utilizadas

Para garantir performance máxima e custo zero de hospedagem, o site foi construído como um **Site Estático**:

* **HTML5 & CSS3:** Estrutura e estilização customizada (sem frameworks pesados).
* **GitHub Pages:** Hospedagem gratuita e automatizada.
* **Custom Domain:** Configurado com domínio `.com` e SSL (HTTPS) forçado.
* **Mobile Friendly:** Layout totalmente responsivo para dispositivos móveis.

---

## 📂 Estrutura de Arquivos

* `index.html`: Estrutura principal e conteúdo do estúdio.
* `style.css`: Toda a lógica visual e design responsivo.
* `/assets`: Pasta contendo logos, artes dos jogos e ícones.
* `CNAME`: Arquivo de configuração necessário para o redirecionamento do domínio personalizado.

---

## 🚀 Como fazer o Deploy

Este projeto utiliza o fluxo de **Continuous Deployment** do GitHub Pages:

1. Qualquer alteração realizada na branch `main` dispara automaticamente um *GitHub Action*.
2. O ambiente de produção é atualizado em segundos após o `git push`.

---

## 🎮 Sobre o Estúdio

O **Shivary** é um estúdio de desenvolvimento indie sediado no Brasil, focado em criar experiências únicas, com títulos já publicados na Steam (como o Foxxel Quest) e novos projetos em desenvolvimento na Unity.
